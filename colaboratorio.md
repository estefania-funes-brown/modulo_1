# Uso de Colab
Google Collab es una herramienta que permite escribir y ejecutar código directamente en el navegador.

Entramos a Google Colab, creamos un "notebook"(cuaderno), apretamos editar y vamos a configuración de notebook, ahí ponemos:

- Tipo de entorno de ejecución R (además tenemos la opción de Python 3 y Julia)
- Acelerador de hardware CPU (tenemos algunas opciones gratis y otras premium)
- Visión de entorno recomendada (última versión).

+Texto para agregar texto sobre el tema que vamos a realizar, lo hacemos utilizando lenguaje técnico

+Código podemos agregar un código de programación. 

Es especial poque no consume recursos, el código se ejecuta en las computadoras de Google, no en la nuestra.
